# zmx2yaml
Methods for parsing Zemax files and creating generalized optical descriptions as YAML files.
The YAML files are formatted to be compatible with the [Batoid Package](https://github.com/jmeyers314/batoid).

## Installation

Pip install from the top level directory using:
`pip install -e .`

## Usage

Use the script `scripts/YAML_from_ZMX.py` to create the file. For documentation and an example see the help in the script by running:

`python scripts/YAML_from_ZMX.py -h`
